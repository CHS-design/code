交叉熵损失速查卡

核心记忆：交叉熵就是取出真实类别的预测概率，再取负对数。

公式

L = -Σi=1C yi log(pi) = -log(pt)

C 是类别数；y_i 是 one-hot 真实标签；p_i 是模型对第 i 类的概率；t 是真实类别编号。

二分类

L = -[y log(p) + (1-y) log(1-p)]

y=1 时，损失是 -log(p)；y=0 时，损失是 -log(1-p)。

为什么重罚“自信地错”

真实类概率 p_t损失 -log(p_t)含义
0.9约 0.105预测对且有信心
0.5约 0.693不确定
0.01约 4.605几乎肯定地预测错

PyTorch

loss = torch.nn.CrossEntropyLoss()(logits, targets)
参数形状与类型注意
logits[N, C]，浮点数原始分数，不要先做 softmax
targets[N]，torch.long类别编号，例如 [2, 0, 1]

常见错误：CrossEntropyLoss 内部已经包含 softmax 与 log；不要写 CrossEntropyLoss()(softmax(logits), targets)。
