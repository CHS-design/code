语义分割双线性上采样速查卡

一句话：双线性插值从空间上的四个邻点按距离加权，适合连续的特征和 logits；离散类别标签只用最近邻插值。

核心公式

P = (1-x)(1-y)F00 + x(1-y)F01 + (1-x)yF10 + xyF11

x,y 是采样点在一个像素小方格中的相对位置，范围为 0 到 1。四项权重相加恒等于 1。

分割张量该怎么缩放

数据典型形状插值方式原因
特征图 / logits[N,C,H,W]bilinear每个值是连续数值；中间估计有意义。
概率图[N,C,H,W]bilinear各通道值是连续概率或分数。
真实类别 mask[N,H,W]nearest0、1、2... 是离散类别编号，不能混合。

PyTorch 模板

import torch.nn.functional as F

# 上采样预测：先放大 logits，后 argmax
logits = F.interpolate(
    logits_small, size=(H, W),
    mode="bilinear", align_corners=False,
)
pred = logits.argmax(dim=1)

# 缩放类别标签：nearest，且恢复 long 类型
mask_small = F.interpolate(
    mask.unsqueeze(1).float(), size=(h, w), mode="nearest"
).squeeze(1).long()

常见错误：不要将类别 mask 用 bilinear 放大。比如类别 0 与类别 2 的边界会生成 0.4、1.2 等无效值。多分类分割也不要先对 logits 做 argmax 再进行双线性插值。

align_corners

它控制缩放前后像素坐标的对齐规则。常用写法是 align_corners=False；训练、验证和推理必须保持一致，避免边界出现细小位移。

学习页：移动采样点，观察权重。关联知识：像素级交叉熵速查卡。
